# Qué es Emotion

Emotion es una librería de estilos en JavaScript basada en CSS-in-JS que se puede utilizar para aplicar estilos a las aplicaciones de React. Proporciona una variedad de funcionalidades para hacer que el proceso de estilado sea más sencillo y potente, como la capacidad de utilizar temas y estilos globales, la integración con el sistema de diseño de su elección y la posibilidad de utilizar estilos basados en propiedades de componentes. Emotion también se centra en la mejora del rendimiento, utilizando técnicas como el cálculo de estilos en tiempo de compilación y la utilización de una caché de estilos para minimizar el tiempo de renderizado.


## Por qué usar Emotion

Hay varias razones por las que alguien podría elegir utilizar Emotion en lugar de otras librerías de estilos para sus aplicaciones de React:

  * Integración con CSS: Emotion le permite utilizar todas las características de CSS, lo que lo hace familiar para los desarrolladores que ya estén familiarizados con esta tecnología.

  * Mejora del rendimiento: Emotion utiliza técnicas como el cálculo de estilos en tiempo de compilación y la utilización de una caché de estilos para minimizar el tiempo de renderizado.

  * Personalización: Emotion proporciona una amplia variedad de opciones de personalización, como la capacidad de utilizar temas y estilos globales, y la posibilidad de utilizar estilos basados en propiedades de componentes.

  * Facilidad de uso: Emotion es fácil de utilizar y no requiere un aprendizaje adicional para aquellos que ya estén familiarizados con CSS.

  * Documentación: Emotion tiene una buena documentación que incluye ejemplos y explicaciones detalladas sobre cómo utilizar cada característica.
